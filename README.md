# Medical-data-history-sql
The Medical Data History Analysis project focuses on analyzing patient healthcare records using SQL to extract meaningful insights from medical datasets. The project demonstrates how Structured Query Language (SQL) can be used to manage, retrieve, and analyze healthcare data efficiently for better decision-making and patient care.
The database contains information related to patients, doctors, diagnoses, treatments, medications, hospital visits, and medical history. Various SQL queries were used to identify trends, generate reports, and answer business and healthcare-related question.
🎯 Project Objectives Retrieve patient and medical history records. Analyze healthcare data using SQL queries. Generate reports based on patient, doctor, and treatment information. Practice SQL concepts through real-world healthcare scenarios. 
🛠️ Technologies Used SQL MySQL MySQL Workbench Git & GitHub 
📚 SQL Concepts Used SELECT WHERE ORDER BY GROUP BY HAVING INNER JOIN LEFT JOIN Aggregate Functions (COUNT, SUM, AVG, MIN, MAX) Subqueries Aliases 
📊 Queries Included This project contains SQL queries for: Retrieving patient records Filtering data based on medical conditions Counting patients and appointments Finding average patient age Analyzing treatment history Doctor-wise patient analysis Sorting and grouping medical records Generating summary reports using aggregate functions 
💡 Skills Demonstrated SQL Query Writing Data Retrieval Data Filtering Data Aggregation Joins and Relationships Database Analysis Problem Solving
